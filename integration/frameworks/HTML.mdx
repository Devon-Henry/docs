## Quick Setup

Add Ragou's AI chat to any HTML website with just 2 lines of code.

### Step 1: Add the Script

Include the Ragou widget script in your HTML `<head>` or before the closing `</body>` tag:

```html
<script src="https://chat-widget-cdn.ragou.workers.dev/app.umd.js"></script>
```

### Step 2: Add the Widget

Place the chat widget element anywhere in your HTML body:

```html
<chat-widget projectId="your-project-id"></chat-widget>
```

<Warning>
Replace `your-project-id` with your actual Project ID from the [Ragou dashboard](https://app.ragou.net/home/teams/projects).
</Warning>

## Complete Example

Here's a minimal HTML page with Ragou chat integrated:

```html
<!DOCTYPE html>
<html lang="en">
<head>
    <meta charset="UTF-8">
    <meta name="viewport" content="width=device-width, initial-scale=1.0">
    <title>My Website</title>
    
    <!-- Ragou Chat Widget Script -->
    <script src="https://chat-widget-cdn.ragou.workers.dev/app.umd.js"></script>
</head>
<body>
    <header>
        <h1>Welcome to My Website</h1>
    </header>
    
    <main>
        <p>Your website content goes here...</p>
    </main>
    
    <footer>
        <p>&copy; 2025 My Website</p>
    </footer>
    
    <!-- Ragou Chat Widget -->
    <chat-widget projectId="your-project-id"></chat-widget>
</body>
</html>
```

## Integration Tips

<AccordionGroup>
  <Accordion icon="code" title="Script Placement">
    **Recommended**: Place the script in the `<head>` section for faster loading.
    
    **Alternative**: Place before closing `</body>` tag to avoid blocking page rendering.
    
    ```html
    <!-- Option 1: In head -->
    <head>
        <script src="https://chat-widget-cdn.ragou.workers.dev/app.umd.js"></script>
    </head>
    
    <!-- Option 2: Before closing body -->
    <body>
        <!-- Your content -->
        <script src="https://chat-widget-cdn.ragou.workers.dev/app.umd.js"></script>
        <chat-widget projectId="your-project-id"></chat-widget>
    </body>
    ```
  </Accordion>
  
  <Accordion icon="mobile" title="Mobile Optimization">
    The widget automatically adapts to mobile screens. Ensure your viewport meta tag is set:
    
    ```html
    <meta name="viewport" content="width=device-width, initial-scale=1.0">
    ```
  </Accordion>
  
  <Accordion icon="shield" title="Content Security Policy">
    If using CSP, allow the Ragou CDN:
    
    ```html
    <meta http-equiv="Content-Security-Policy" 
          content="script-src 'self' https://chat-widget-cdn.ragou.workers.dev;">
    ```
  </Accordion>
</AccordionGroup>

## What Happens Next

1. **Widget Appears**: A floating chat button appears in the bottom-right corner
2. **Click to Open**: Users click the button to open the chat window  
3. **AI Responds**: Your trained AI answers questions based on your uploaded documents
4. **Fallback Support**: Forms are generated for questions the AI cannot answer
